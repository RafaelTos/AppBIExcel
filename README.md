# 📊 Dashboard Interativo de BI - Excel Automation

Aplicação desktop de *Business Intelligence* (BI) desenvolvida em Python para automatização, leitura e visualização dinâmica de bases de dados do Excel através de uma interface gráfica interativa.

---

## 🚀 Funcionalidades

* **Leitura Inteligente:** Carrega automaticamente a primeira aba de qualquer ficheiro Excel e mapeia as colunas dinamicamente.
* **Filtros Estilo BI:** Segmentação de dados na interface gráfica com atualização do gráfico em tempo real.
* **Lógica Top N + Outros:** Agrupamento inteligente automático para tratar colunas com muitos dados (alta cardinalidade), mantendo o visual limpo, legível e profissional.
* **Controlo Total:** Barra de ferramentas integrada para dar zoom, mover e exportar os gráficos como imagem (PNG/JPEG).
* **Margens Corrigidas:** Ajuste dinâmico que impede que etiquetas ou textos longos apareçam cortados no eixo X.

---

## 🛠️ Tecnologias Utilizadas

* **Interface (Front-End):** `Tkinter`
* **Processamento de Dados:** `Pandas` | `OpenPyXL`
* **Visualização de Dados:** `Matplotlib`
